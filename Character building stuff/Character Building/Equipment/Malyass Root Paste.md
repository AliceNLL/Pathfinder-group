# Malyass Root Paste

**Stage 1** [[Clumsy 1]] and -10-foot status penalty to all Speeds (1 minute)

**Stage 2** [[Clumsy 1|Clumsy 2]] and -20-foot status penalty to all Speeds (1 minute)
